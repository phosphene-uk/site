# phosphene.uk

The Phosphene Studio landing page — a single static HTML page, deployed to Cloudflare Pages.

- `index.html` — the page (wordmark, gloss, contact, definition mark)
- `_headers` — security headers applied by Cloudflare Pages

No build step. Edit `index.html`, commit, and push; Cloudflare Pages redeploys automatically.
